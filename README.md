# Velhac — Mobile Dev & Hacking Toolkit for Termux

> By [Smart Tech Programming](https://youtube.com/@smarttechprogramming)

Velhac is an all-in-one Termux tool installer for Android, combining **hacking/security tools** and **mobile dev tools** in one clean menu. No laptop needed.

---

### Installation

```bash
pkg update
pkg install wget -y
wget -O $PREFIX/bin/velhac https://raw.githubusercontent.com/edunoluwadarasimidavid/velhac/main/velhac.sh
chmod +x $PREFIX/bin/velhac
```

### Launch

```bash
velhac
```
